# agnostic-http-errors

A Modern Http Errors.

## `Installation`

```bash
# npm
$ npm install agnostic-http-errors
# yarn
$ yarn add agnostic-http-errors
```

## `Usage`

```typescript
import HttpErrors from "agnostic-http-errors";

throw new HttpError(404, "Not Found!");
```
